### **1\. Capaian Pembelajaran Lulusan (CPL) yang Dibebankan**

* **Sikap (S9):** Menunjukkan sikap bertanggung jawab atas pekerjaan di bidang keahliannya secara mandiri.  
* **Pengetahuan (P2):** Menguasai konsep dan prinsip rekayasa perangkat lunak serta bahasa pemrograman berbasis web.  
* **Keterampilan Umum (KU2):** Mampu menunjukkan kinerja mandiri, bermutu, dan terukur.  
* **Keterampilan Khusus (KK3):** Mampu merancang dan membangun aplikasi web dinamis dengan mengintegrasikan basis data dan kerangka kerja (*framework*).

### **2\. Capaian Pembelajaran Mata Kuliah (CPMK)**

1. **CPMK-1:** Mahasiswa mampu menjelaskan arsitektur web, protokol HTTP, dan konsep *Client-Server*.  
2. **CPMK-2:** Mahasiswa mampu membangun antarmuka web yang responsif menggunakan HTML5, CSS3, dan *utility-first CSS* (Tailwind/Bootstrap).  
3. **CPMK-3:** Mahasiswa mampu mengimplementasikan logika interaktif pada sisi klien menggunakan JavaScript (DOM Manipulation).  
4. **CPMK-4:** Mahasiswa mampu mengembangkan aplikasi web dinamis menggunakan bahasa sisi server (PHP/Node.js) dan integrasi basis data (MySQL).  
5. **CPMK-5:** Mahasiswa mampu mengaplikasikan pola desain MVC menggunakan *framework* modern untuk membangun proyek aplikasi web yang aman dan skalabel.
