## 🎯 Capaian Pembelajaran Mata Kuliah (CPMK)

| CPMK | Kemampuan |
|------|-----------|
| CPMK-1 | Menjelaskan arsitektur web, protokol HTTP, dan konsep Client-Server |
| CPMK-2 | Membangun antarmuka web yang responsif menggunakan HTML5, CSS3, dan utility-first CSS |
| CPMK-3 | Mengimplementasikan logika interaktif pada sisi klien menggunakan JavaScript |
| CPMK-4 | Mengembangkan aplikasi web dinamis menggunakan PHP dan integrasi basis data MySQL |
| CPMK-5 | Mengaplikasikan pola desain MVC menggunakan framework modern |

## 🛠️ Tools & Software yang Diperlukan

- **Text Editor**: Visual Studio Code
- **Browser**: Google Chrome / Mozilla Firefox (with DevTools)
- **Server Lokal**: XAMPP / Laragon / Herd
- **Database**: MySQL (via phpMyAdmin)
- **Version Control**: Git & GitHub

## 📝 Cara Menggunakan

1. Ikuti urutan materi sesuai minggu perkuliahan
2. Praktikkan setiap contoh kode secara langsung
3. Kerjakan latihan soal di akhir setiap bab
4. Bangun proyek kecil untuk mengintegrasikan setiap topik
